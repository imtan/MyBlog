これは縦書きの **Markdown** 記事です。
改行はパラグラフに変換されます。

```js
// これはコードブロック。横書きで表示されます。
function greet() {
  console.log("こんにちは、世界！");
}
```

後半も縦書きで続きます。現代のウェブデザインにおいても、縦書きは独特の美しさを持っています。

```js
// 日本語の変数名と関数名
const 挨拶 = "こんにちは";
function 表示する(メッセージ) {
  console.log(メッセージ);
}
表示する(挨拶);
```

しかし、コメントやドキュメントには積極的に日本語を使うことで、理解しやすくなります。
